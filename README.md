# Business Intelligence Lab

POC evaluation workspace for analytics dashboard tools.

## Objective
Evaluate Grafana, Metabase, and other BI tools for generating insights from SQL, NoSQL, and time-series databases populated by Dagster pipelines.

## Quick Start
1. Clone repository
2. Run `./scripts/setup/bootstrap-tools.sh`
3. Access tools at documented ports
4. Review `/docs/evaluation-criteria.md`

## Structure
- `/dashboards` - Tool-specific dashboard definitions
- `/data-sources` - Database connectors and schemas
- `/infrastructure` - Deployment configurations
- `/evaluation` - Comparative analysis artifacts
